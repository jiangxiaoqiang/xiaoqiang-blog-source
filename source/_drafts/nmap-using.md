#### 查看指定端口的状态

有时我们想查看服务器具体的某一个端口是否UP，可以通过扫描实现。但是还有一种更加节省时间的方法，例如想查看凤凰网的80端口是否开启，可以输入如下命令：

```Bash
nmap -p 80 ifeng.com
```

结果可以立即出来，这样可以节省不少时间。p参数用于监控指定端口。

#### 隐藏身份

当我们在用nmap扫描时，一般情况下不建议直接扫描，直接扫描会暴露自己的IP，如果需要隐藏身份，可以有如下方法：

* 伪造访问的IP地址
* 对发送信息进行处理
* 风险进行嫁接
* 其他
