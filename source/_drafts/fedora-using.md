#### 安装MPlayer

输入如下命令安装MPlayer：

```Bash
#安装视频播放器MPlayer
dnf install -y mplayer
```

安装完毕后在终端中输入如下命令即可播放：

```Bash
#播放文件
mplayer Heidi.2015.海蒂和爷爷.720p.Chi_Ger.ZMZ-BD-MP4.mp4

#mplayer命令加文件名
mplayer filename
```

#### 磁盘分析器

从终端里启动磁盘使用分析器，只需输入Baobab，然后按回车键。

```Bash
#启动磁盘分析器
baobab
```
