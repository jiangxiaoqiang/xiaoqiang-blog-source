使用的Ubuntu版本为14.04 LTS和16.04 LTS版本。

#### 小技巧

* 长按Windows键会出现全局快捷键页面

#### 快捷键

| 快捷键 | 作用 |
| ----------------- |:-------------:|
| Ctrl + Alt + + | 放大终端 |
| Ctrl + - | 缩小终端 |
| Ctrl + Window + Up | 最大化窗口，注意按下时按顺序按下相应键 |
| Ctrl + Window + Down | 最小化窗口，注意按下时按顺序按下相应键 |
| Alt + F1 | 调出侧边栏(如果有设置自动隐藏的话) |
