#### 表示美玉的汉字

<!-- more -->

| 字    | 含义                                       |
| ---- | ---------------------------------------- |
| 琮    | 瑞玉。方柱形，中有圆孔。用为礼器、贽品、符节等。《周礼·春官·大宗伯》：“以玉作六器，以礼天地四方，以苍璧礼天，以黄琮礼地。” |
| 瑷    | 《玉篇·玉部》：“瑷，美玉也。”                         |
| 琼    | 《诗·卫风·木瓜》：“投我以木瓜，报之以琼琚。”                 |



#### 生僻字


| 字    | 含义                                       |
| ---- | ---------------------------------------- |
| 犇    | 犇（读音bēn），本意指牛惊走。引申泛指奔跑，此意后作“奔”。《第一批异体字整理表》将“犇”归为“奔”的异体字，但是只用于“bēn”音。此字也多用于人名。 |
| 羱 | 读作yuán，指一种大角羊。〔～羊〕古书上说的一种大角羊。亦称“北山羊”。 |
